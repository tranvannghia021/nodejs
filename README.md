# nodesj basic

App manager courses.
Simple app created with nodejs and handlebars.

## Project setup

```bash
  npm install
```

### Compiles and hot-reloads for development

```bash
  npm run serve
```

### Compiles and minifies for production

```bash
  npm run build
```

## Tech Stack

**Client:** Handlebars,jquery,scss

**Server:** Nodejs, MongoDB...
